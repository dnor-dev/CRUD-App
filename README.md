# CRUD-App
CRUD App with Database 

APIS
======

For creating the payload
POST Requests
/apis/zuridata/create
 

For getting the data
GET Requests
/apis/zuridata/


For updating the data
PUT Requests
/apis/zuridata/:id


For deleting the data
DELETE Requests
/apis/zuridata/:id
